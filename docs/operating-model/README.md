# Operating Model Documents

This folder is the canonical shared layer for agent behavior.

Read order:

1. `AGENT_OPERATING_MODEL.md`
2. `SKILL_CATALOG.md`
3. `AGENT_EVALUATION_CHECKLIST.md`
4. Existing workflow rules such as `dynamic-routing.md`, `role-definitions.md`, `quality-gates.md`, and `handoff-contract.md`

Recommended rule:

- Before routing work, read the operating model.
- Before selecting a skill, read the skill catalog.
- Before completing work, run the evaluation checklist.
